## **Configuration Details**

### **Files Modified for Deployment:**

1. **vite.config.ts** - Set base path
   ```typescript
   export default defineConfig({
     base: '/anatomy/',  // Required for subpath deployment
     plugins: [react()],
     server: {
       port: 5173,
       open: true
     }
   })
   ```

2. **src/App.tsx** - React Router setup
   ```typescript
   function App() {
     return (
       <BrowserRouter basename="/anatomy">  // Required for routing
         <Routes>
           <Route path="/" element={<HomePage />} />
           <Route path="/learn/:anatomyId" element={<LearnPage />} />
           <Route path="/editor/:anatomyId" element={<EditorPage />} />
         </Routes>
       </BrowserRouter>
     );
   }
   ```

3. **.github/workflows/deploy.yml** - GitHub Actions workflow
   - Builds on every push to `main`
   - Deploys to `gh-pages` branch automatically
   - Uses official GitHub Pages actions

## **Troubleshooting**

### **Site shows blank/white page:**
1. Check browser console (F12) for errors
2. Verify `basename="/anatomy"` is set in `src/App.tsx`
3. Check that `base: '/anatomy/'` is set in `vite.config.ts`
4. Force refresh: `Ctrl+Shift+R`

### **Deployment failed:**
1. Go to: https://github.com/liorcohencell-alt/anatomy/actions
2. Click the failed run
3. Check logs for error details
4. Common issues:
   - TypeScript errors in code
   - Build failures
   - Missing dependencies

### **Changes not appearing:**
1. Verify push was successful: `git log --oneline -1`
2. Check Actions page for workflow status
3. Wait 2-3 minutes for deployment to complete
4. Hard refresh browser: `Ctrl+Shift+R`
